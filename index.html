En este codelab, aprenderás cómo instalar tu primera proxi en Linux.

  1. Ejecutar el siguiente comando en la terminal para instalar Squid3:
   sudo apt-get install squid3
  1. Verificar la instalación:
   squid -v
  1. Actualizar la lista de paquetes disponibles:
   sudo sudo apt-get update
  1. Descargar archivo Squid reconfigurado: squid.conf
  1. Abrir el archivo de configuración squid.conf que se descargo en el paso anterior:
   sudo gedit [agrega-tu-ruta-de-descarga]
  1. Configurar las líneas: Línea 6041: Reemplazar con tu nombre de usuario (echo $USER) Línea 1009: Reemplazar con tu dirección IP (ifconfig) Línea 1209: Reemplazar con tu dirección IP

Guardar y cerrar el archivo
  1. Ejecutar el siguiente comando en la terminal para instalar Squid3:
   cd /etc/squid
   sudo mv squid.conf squid.conf.old
   sudo cp /path/to/squid.conf /etc/squid
   sudo mkdir /etc/squid/listas
   cd listas
   sudo touch /etc/squid/listas/denegados
   sudo gedit denegados
  1. En el archivo denegados, agregar las URLs que deseas bloquear (por ejemplo, facebook.com, youtube.com)
  2. Guardar y cerrar el archivo
  3. Reiniciar el servicio Squid:
   sudo /etc/init.d/squid restart
  1. Ir a Settings > Network > Network Proxy
  2. Seleccionar Manual
  3. Ingresar los valores del proxy configurado en squid.conf
  1. Abrir un navegador web
  2. Intentar cargar las URLs bloqueadas (por ejemplo, facebook.com)
  3. Si la configuración fue exitosa, las URLs bloqueadas no deberían cargarse

¡Felicitaciones! Has instalado tu primera proxi en Linux
